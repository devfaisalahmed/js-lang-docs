# Primitive Datatype

- **string**
  ```javascript
     var str = "Faisal ahmed";
  ```
- **number**

  ```javascript
    var number = 10;
  ```  
- **boolean**
  ```javascript 
   var a = true/ false;
    ```
- **undefined**
  ```javascript 
  var a; --যার মান নিরধারন করা হই নি 
    ```
- **null**
  ```javascript 
  var a = null;
    ```
