# Primitive Datatype

| name      | value                               |
| --------- | ----------------------------------- |
| String    | ` var str = "Faisal ahmed";`        |
| Number    | ` var number = 10;`                 |
| Boolean   | `var a = true/ false;`              |
| Undefined | ` var a; যার মান নিরধারন করা হই নি` |
| Null      | ` var a = null;`                    |
